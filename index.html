☀️ Instruções da Missão Galáctica

Objetivo: Organize as cartas espaciais em dois grupos corretos e na sequência adequada.

🟢 Grupo 1: Consultar Produto

Arraste as cartas relacionadas à consulta de produtos para este grupo e organize na ordem correta do processo.

🟡 Grupo 2: Cancelar Produto

Arraste as cartas relacionadas ao cancelamento de produtos para este grupo e organize na ordem correta do processo.

Regras Importantes:

  • Você possui 5 vidas espaciais ⭐
  • Cada erro na verificação remove uma vida
  • Cartas no grupo errado serão devolvidas automaticamente
  • Cartas na ordem errada ficarão vermelhas para te ajudar
  • Use os botões ▲▼ para reorganizar as cartas dentro dos grupos
  • Use o botão ✕ para remover cartas dos grupos
  • Complete a missão para receber seu certificado galáctico!

Como Jogar:

  1. Arraste as cartas para os grupos corretos
  2. Organize as cartas na sequência correta dentro de cada grupo
  3. Clique em "🚀 Verificar Missão" para validar sua resposta
  4. Ajuste conforme o feedback recebido
  5. Complete ambos os grupos corretamente para vencer!
